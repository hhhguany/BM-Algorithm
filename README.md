# BM-Algorithm
Berlekamp-Massey 算法的 Python 3 实现。

## 使用
```
$ python main.py -h
usage: main.py [-h] [-d {1,2,3}] [-l {ch,en}] [-v VERSION] flow
```

### 普通模式
```
$ python main.py 10011011000111010100
输入序列为：10011011000111010100
可以生成此序列的最优为 11 级线性反馈移位寄存器
该寄存器的连接多项式为：f(x) = 1 + x^3 + x^9 + x^11
```
